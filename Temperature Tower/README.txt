# Summary

I liked XodrateP's temperature tower, but needed one for PETG, so I changed the numbers to suit PETG temps. You'll still need to slice it accordingly in S3D.

Edit: Don't go all the way to 260 if you don't have an all-metal heatbreak, or it will melt your PTFE tube.

Printed at 0.2mm layer high, 10% infill.
