# Summary

Designed for 1/2" thick material and the holes 3/16th. 

Depending on the keycap the holder ranges from loose to a little tight.

I kept on having the display mounts falling over so I connected them in pairs of fours. I'm still new to 3D printing so I'm sure there's a better solution.
